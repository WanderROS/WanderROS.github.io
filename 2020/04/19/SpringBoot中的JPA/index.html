SpringBoot中的JPA

发表于 2020-04-19 | 分类于 Java

简介

JPA(Java Persistence API)是Sun公司提出的一种Java持久化规范。使得Java开发人员可以以一种对象/关系映射的方式来管理Java应用中的关系数据。

  • 持久化（Persistence）：把数据保存到可以永久存储的设备中。JDBC就是一种持久化机制；文件I/O也是一种持久化机制。
  • 规范：明文规定或者约定俗成的标准。
  • Hibernate是一个开源的对象关系映射框架，对JDBC进行了非常轻量级的对象封装，将POJO与数据库表建立映射关系，能自动生成SQL语句，自动执行。
  • ORM（Object Relational Mapping）：用于实现面向对象编程语言里不同系统类型的系统之间的数据转换。
  • Spring Data：用于简化数据库访问，并支持云服务的开源框架。
