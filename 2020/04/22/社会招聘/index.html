社会招聘

发表于 2020-04-22 | 分类于 Work

硬性要求

  1. 全日制统招本科生以上，不包含三本；
  2. 工作经验5年以上；
  3. 以前在集团工作过的不予考虑。

通用要求

  1. 具备良好的学习能力、逻辑思维能力、沟通能力、团队协作能力和执行力；
  2. 具有良好规范的文档和编程习惯，具备良好的沟通能力。

算法工程师

资格

  1. 硕士以上；
  2. 扎实的数据结构和算法设计基础，掌握C/C++/Python/Perl等一门以上脚本语言；
  3. 对基础数据进行整理清洗和数据挖掘，利用统计分析、回归、决策树、神经网络等技术建立模型，参与各类模型的设计、开发、实施与跟踪，结合实际效果不断进行模型的训练和迭代，管理模型生命周期的全流程，挖掘用户画像和设备画像；
  4. 熟悉主流机器学习或深度学习，如Sklearn、Spark MLlib、Tensorflow等，熟练使用SQL，熟悉主流数据库环境；
  5. 熟悉图像处理、模式识别、图像滤波、轮廓提取、边缘检测、图像分割、图像拼接、图像融合等各类图像处理理论者优先；
  6. 熟悉语音信号处理、语言模型、声纹识别、模式识别、ASR、NLU、TTS等语音识别技术者优先；
  7. 地点：上海

职责

  1. 从事利用深度学习技术研发实现图像识别和机器视觉的功能应用相关研究工作；
  2. 负责图像处理、图像特征提取、模型训练，识别衣服材质、件数、种类、颜色等；
  3. 负责ASR、NLU、TTS等语音识别技术实现。

后端开发工程师

资格

  1. 扎实的Java语言功底，熟悉JVM，熟练使用Jenkins、Maven、Git等常用工具；
  2. 熟悉springMvc、mybatis、hibernate、Docker、Kubernetes等开发框架；
  3. 熟悉常用的微服务框架（Dubbo/SpringCloud）以及其它常见的开源框架；
  4. 熟悉分布式缓存、消息中间件、RPC框架、负载均衡等技术，掌握各类中间件使用，包括Nginx、Tomcat、Redis、kafka等；
  5. 熟悉Linux操作系统，掌握常用的Linux命令；
  6. 有家电互联网经验的优先；
  7. 地点：上海

职责

  1. 负责云端框架设计和搭建；
  2. 负责应用的微服务开发，满足产品功能需求；
  3. 负责跟IoT、前端、第三方供应商接口对接；
  4. 负责云端并发性、可靠性、容错性、稳定性、负载均衡等设计。

大数据工程师

资格

  1. 硕士以上；
  2. 熟悉常用设计模式，掌握分布式服务、高扩展，高可用，高性能系统的设计与编码，大规模存储系统、实时及离线计算、数据仓库方面有经验者优先；
  3. 熟悉大数据生态包括不限于 Hadoop/Spark/Hive/Hbase/Yarn/ClickHouse/ElasticSearch,并有相关项目开发经验；
  4. 熟悉流计算编程，熟练掌握Flink、Kafka、Flume等开源工具或框架；
  5. 熟悉数据ETL的流程和每个过程的关键数据处理方法和技巧，对OLAP、报表及可视化有一定的了解；
  6. 熟练使用Java/Python/SQL等编程语言，具备良好软件敏捷开发习惯；
  7. 地点：上海

职责

  1. 负责大数据平台设计和搭建；
  2. 负责从海量日志提取数据和数据分析，支持市场以及营销活动的数据需求；
  3. 负责用户画像分析；
  4. 负责设备画像分析；
  5. 根据业务目标进行数据建模、数据挖掘，学习、研究Hadoop及相关技术内核，并将其运用与实践；

测试工程师

资格

  1. 5年以上互联网产品功能测试工作经验；
  2. 熟悉软件测试基本理论，掌握常用测试工具、软件测试流程及其各项规范，能独立承担测试工作（MONKEYRUNNER、DDMS、LR、SOAP等工具）；
  3. 有自动化测试经验，能独立设计用例并编写代码实现自动化测试，有产品性能、安全等测试经验；
  4. 很强的学习能力、分析能力和解决问题的能力，熟悉持续集成理论并有过相关实施经验，能够熟练使用持续集成平台及工具；
  5. 有家电互联网测试工作经验优先;
  6. 地点：上海

职责

  1. 负责智能产品系统测试、如模块、app、云平台等的功能，性能测试；
  2. 参与自动化开发及测试；
  3. 参与测试技术规范和品质标准的制定。

终端软件工程师

资格

  1. 具有C/C++程序开发经验，熟悉Socket网络编程、文件I/O操作、音视频、多进程、多线程开发；
  2. 主要是界面和功能开发，使用工具Qt和miniGUI；
  3. 熟悉linux内核裁剪和定制、驱动开发、应用程序移植者优先；
  4. 熟悉shell脚本者优先；
  5. 深刻理解处理器架构，热爱处理器相关技术。深刻理解内存、DMA、中断等概念；
  6. 地点：无锡

职责

  1. 嵌入式软件平台架构设计和技术解决方案提供；
  2. 带屏嵌入式系统UI开发和驱动开发；
  3. 组织软件沟通，评审，检讨；
  4. 从事智能产品开发工作，提高软件稳定性和安全性。

终端硬件工程师

资格

  1. 有较强的模拟/数字电路的硬件设计和调试能力，经历过从单元板研发到整机产品研发到正式交付完整的研发周期；
  2. 负责产品硬件设计、器件选型、原理图设计和PCB Layout、硬件调试与设计迭代，有多层板设计经验者优先；
  3. 熟悉显示屏、扬声器、麦克风、摄像头、底盘马达、红外和LED等外设或传感器硬件；
  4. 撰写相关的标准, 流程，能够独立搭建测试环境与测试系统，能熟练运用示波器、逻辑分析仪、信号发生器等；
  5. 地点：无锡

职责

  1. 负责硬件设计、器件选型、PCB layout、调试等；
  2. 负责标准化、硬件降本、wifi模组降本；
  3. 负责为各个模块选择合适的处理器平台及传感器；
  4. 对各类处理器平台和传感器进行比较分析（成本、性能、能耗、稳定性、兼容性），提交可行性报告。

其他要求

  1. 简历名称格式：葛先生-34岁-6年经验-机器视觉工程师-猎聘简历
  2. 一个岗位一周安排面试数要达到5个左右，但是不能超过5个人；
