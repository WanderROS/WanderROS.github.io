开始

基础知识

  1. Mac上安装Python3（安装的时候版本已经是3.7.7了）：

  2. 使用的时候可以在终端中输入python3进入命令行环境；

  3. Python解释器读取整个程序，确定其中每个单词的含义，程序存在严重的错误时，Python将显示traceback；

  4. 变量存储了一个与变量相关联的信息，在程序中可随时修改变量的值，而Python将始终记录变量的最新值；

    • 变量名只能包含字母、数字和下划线
    • 变量名不能包含空格，但可使用下划线来分隔其中的单词
    • 不要将Python关键字和函数名用作变量名
    • 变量名应既简短又具有描述性
    • 慎用小写字母l和大写字母O，因为它们可能被人错看成数字1和0

  5. 在Python中，用引号括起的都是字符串，其中的引号可以是单引号，也可以是双引号；

  6. 方法是Python可对数据执行的操作,每个方法后面都跟着一对括号,因为方法通常需要额外的信息来完成其工作;

  7. Python使用加号（+）来合并字符串,称为拼接，可使用存储在变量中的信息来创建完整的消息；

  8. 空白泛指任何非打印字符，如空格、制表符和换行符；

  9. Python能够找出字符串开头和末尾多余的空白，要确保字符串末尾没有空白，可使用方法rstrip()，还可以剔除字符串开头的空白，或同时剔除字符串两端的空白，分别使用方法lstrip()和strip()，要永久删除这个字符串中的空白，必须将删除操作的结果存回到变量中；

  10. 如果需要在一个函数内修改全局变量，就使用 global 语句

  11. 在Python中，可对整数执行加（+）减（-）乘（）除（/）运算，将带小数点的数字都称为*浮点数**，调用函数str()将非字符串值表示为字符串；

  12. 在Python中，注释用井号（#）标识，井号后面的内容都会被Python解释器忽略；

  13. 列表由一系列按特定顺序排列的元素组成，在Python中，用方括号（[]）来表示列表，并用逗号来分隔其中的元素，列表是有序集合，因此要访问列表的任何元素，只需将该元素的位置或索引告诉Python即可，通过将索引指定为-1，可让Python返回最后一个列表元素，也适用于其他负数索引，例如，索引-2返回倒数第二个列表元素；

    1. 方法append()将元素添加到了列表末尾，而不影响列表中的其他所有元素

    2. 方法insert()可在列表的任何位置添加新元素，需要指定新元素的索引和值

    3. 知道要删除的元素在列表中的位置，可使用del语句删除元素

    4. 方法pop()可删除列表末尾的元素，并让你能够接着使用它，可以使用pop()来删除列表中任何位置的元素，只需在括号中指定要删除的元素的索引即可

    5. 方法remove()可以只知道要删除的元素的值便可删除元素，只删除第一个指定的值

  14. Python方法sort()让你能够较为轻松地对列表进行永久性排序，可以按与字母顺序相反的顺序排列列表元素，只需向 sort() 方法传递参数reverse=True；

  15. 要保留列表元素原来的排列顺序，同时以特定的顺序呈现它们，可使用函数sorted()，也可向函数sorted()传递参数reverse=True；

  16. 方法reverse()反转列表元素的排列顺序,注意也是永久性的修改列表中元素的排列顺序，想要恢复再次reverse()；

  17. 使用函数len()可快速获悉列表的长度：

  19. Python函数 range() 让你能够轻松地生成一系列的数字，从指定的第一个值开始数，并在到达你指定的第二个值后停止生成数字列表，可使用函数list()将range()的结果直接转换为列表，允许指定步长，默认是1：

  21. 可以轻松地找出数字列表的最大值、最小值和总和，分别使用max、min、sum函数；

  22. 列表解析将for循环和创建新元素的代码合并成一行，并自动附加新元素，要使用这种语法，首先指定一个描述性的列表名，如squares；然后，指定一个左方括号，并定义一个表达式，用于生成你要存储到列表中的值：

  23. 处理列表的部分元素——Python称之为切片，要创建切片，可指定要使用的第一个元素和最后一个元素的索引，注意指定的是第几个元素开始，索引结束之前的元素为止，没有指定第一个索引，Python将自动从列表开头开始，要让切片终止于列表末尾，也可使用类似的语法，省略终止索引，都不指定类似于不使用[]，负数索引返回离列表末尾相应距离的元素：

元组

  1. Python将不能修改的值称为不可变的，而不可变的列表被称为元组；
  2. 元组看起来犹如列表，但使用圆括号而不是方括号来标识，可以使用索引来访问其元素，就像访问列表元素一样；
  3. 像列表一样，可以使用for循环来遍历元组中的所有值；
  4. 虽然不能修改元组的元素，但可以给存储元组的变量赋值；
  5. 为什么单独列出元组，因为元组可以作为函数的返回值！

判断

  1. if语句让你能够检查程序的当前状态，并据此采取相应的措施；

  2. 在Python中检查是否相等时区分大小写，如果大小写无关紧要，而只想检查变量的值，可将变量的值转换为小写，再进行比较；

  3. 要判断两个值是否不等，可结合使用惊叹号和等号（!=），其中的惊叹号表示不，使用两个等号（==）判断两个值是否相等；

  4. 使用and检查多个条件，使用or检查多个条件，前者必须全部都符合，后者只需要有一个通过就符合；

  5. 关键字 in 判断特定的值是否已包含在列表中，关键字not in 判断是否不在列表中；

  6. Python只执行if-elif-else结构中的一个代码块，它依次检查每个条件测试，直到遇到通过了的条件测试；

字典

  1. 在Python中，字典是一系列键—值对，每个键都与一个值相关联，可以使用键来访问与之相关联的值，字典用放在花括号{}中的一系列键—值对表示，键和值之间用冒号分隔，而键—值对之间用逗号分隔：

  2. 字典是一种动态结构，可随时在其中添加键—值对：

  3. 在空字典中添加键—值对是为了方便,直接使用{}可以创建空字典；

  4. 字典中不再需要的信息，可使用del语句将相应的键—值对彻底删除，使用del语句时，必须指定字典名和要删除的键；

  5. 用于遍历字典的for循环，可声明两个变量，用于存储键—值对中的键和值，对于这两个变量，可使用任何名称：

  6. Python不关心键—值对的存储顺序，而只跟踪键和值之间的关联关系；

  7. 方法keys()返回字典中的所有键；

  8. 方法values()返回一个值列表；

  9. 将一系列字典存储在列表中，或将列表作为值存储在字典中，这称为嵌套；

用户输入

  1. 函数input()让程序暂停运行，等待用户输入一些文本，获取用户输入后，Python将其存储在一个变量中,使用函数input()时，Python将用户输入解读为字符串；

  2. 函数int()将数字的字符串表示转换为数值表示;

  3. 求模运算符（%）将两个数相除并返回余数;

类

  1. 在Python中，首字母大写的名称指的是类（约定）；

  2. 方法init()是一个特殊的方法 ，每当你根据类名创建新实例时， Python都会自动运行它 ，在这个方法的名称中，开头和末尾各有两个下划线，自动传入实参self ，每个与类相关联的方法调用都自动传递实参self，它是一个指向实例本身的引用，让实例能够访问类中的属性和方法；

  3. 根据类名创建实例后，就可以使用句点表示法来调用类中定义的任何方法；

  4. 一个类继承另一个类时，它将自动获得另一个类的所有属性和方法；原有的类称为父类，而新类称为子类 ;

  5. 子类继承了其父类的所有属性和方法，同时还可以定义自己的属性和方法 ;

  6. 创建子类的实例时， Python首先需要完成的任务是给父类的所有属性赋值，子类的方法init()需要父类施以援手；

  7. super()是一个特殊函数，帮助Python将父类和子类关联起来，父类也称为超类（ superclass），名称super因此而得名 ；
