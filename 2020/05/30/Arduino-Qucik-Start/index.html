Arduino Quick Start

发表于 2020-05-30 | 分类于 Arduino

简介

Arduino 是世界上最成功（有些人可能对此持有异议）的开源硬件项目之一。Arduino 团队公开其设计之后，包括意大利、巴西、中国、荷兰、印度和美国在内的很多国家，都在生产基于这种设计的开发板！Arduino 面向的是艺术家和热衷于捣鼓东西的人，而不是那些对技术细节感兴趣以及痴迷于项目规划、技术规范与操作说明书的工程师，但是还有一款黑客级别的Teensy项目也是基于arduino项目的，这一点，让我特别关注arduino！推荐书籍：《Arduino 技术指南 》· [美] John M. Hughes 著 武传海 译

开始

Arduino

  1. Arduino 的诞生源于Massimo Banzi 与 David Cuartielles 等人想开发这样一种简单设备： 能够很容易地连接到其他各种设备上，应当很容易进行编程，并且价格也要便宜；

  2. Arduino 每年都在不断推出新型开发板，新型 Arduino 开发板会采用更强大的处理器，拥有更大内存，还配有性能大幅增强的输入 / 输出（I/O）功能，但它们大部分都会采用相同的引脚排列，并且都与现有的扩展板（shield）以及各种扩展组件（比如传感器、继电器、驱动器）协同工作；

  3. Arduino 的核心是处理器与预安装的 Bootloader；

  4. Arduino 开发板不能使用微控制器提供的所有功能；

  5. Arduino BootLoader 实现了一种特殊的协议，该协议让 BootLoader 识别出 Arduino IDE，并允许用户把程序数据从开发主机传送到目标开发板；

arduino库

  1. Arduino IDE 自身带有一系列函数库，方便用户在程序中使用；
  2. EEPROM 库用于支持对 AVR 内置 EEPROM 的读写操作，并可实现永久性存储；
  3. Ethernet 库提供了与 Arduino Ethernet 扩展板进行交互的必需功能 ；
  4. Firmata 库提供了使用类似 MIDI 的协议，使得在 Arduino 与主机应用之间进行串行通信成为可能；
  5. GSM 库和 GSM 扩展板一起用于连接到 GSM/GPRS 网络；
  6. LiquidCrystal库允许 Arduino 开发板控制液晶显示器（LCD）模块；
  7. SD 库对读写 SD 闪存卡提供支持，包含全尺寸与微型 SD 类型；
  8. Servo 库提供一系列函数，用于控制伺服电机，比如遥控飞机中的那些电机 ；
  9. SPI 库支持使用串行外围接口（SPI）总线，以便与兼容 SPI 的外围设备进行通信；
  10. SoftwareSerial 库在 Arduino 的数字 I/O 引脚上实现了基于软件的串行通信 ；
  11. Wi-Fi 库让 Arduino 拥有连接无线网络的能力；
  12. 针对 Arduino 开发板，有许多第三方库可以使用！
