  • Bing图像接口：

    • 地址：https://cn.bing.com/HPImageArchive.aspx?n=1

    • 支持参数：

      • n必填，返回照片数量
      • idx非必填，用于指定获取哪天的壁纸，0表示当天，1表示昨天，2表示前天，依此类推
      • format非必填，用于指定输出格式，默认是 XML 的，传入 js 表示返回 JSON 格式
      • mkt非必填，用于指定推送地区，如：en-US，zh-CN，ja-JP，en-AU，en-UK，de-DE，en-NZ，en-CA

    • 取参数中的url字段内容，拼接到https://bing.com/后面形成完整的图片url。
