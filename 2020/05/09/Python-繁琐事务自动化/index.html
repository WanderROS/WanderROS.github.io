模式匹配和正则表达式

  1. Python 中所有正则表达式的函数都在 re 模块中，在使用的时候需要导入模块；

  2. 向 re.compile()传入一个字符串值，表示正则表达式，将返回一个Regex模式对象；

  3. Regex 对象的 search()方法查找传入的字符串，寻找该正则表达式的所有匹配：

    • 没有找到匹配该正则表达式时返回None；
    • 找到了则返回一个Match对象，Match对象有一个group()方法，返回被查找字符串中实际匹配的文本；

  4. 添加括号将在正则表达式中创建“分组”,想要一次就获取所有的分组，请使用 groups()方法：

  5. 字符|称为“管道”，希望匹配许多表达式中的一个时，就可以使用它；

  6. 字符?表明它前面的分组在这个模式中是可选的，即可以出现零次或者一次；

  7. *（称为星号）意味着“匹配零次或多次”，即星号之前的分组，可以在文本中出现任意次；

  8. +（加号）意味着“匹配一次或多次”；

  9. 如果想要一个分组重复特定次数，就在正则表达式中该分组的后面，跟上花括号包围的数字，比如(Ha){3,5}；

  10. Python 的正则表达式默认是“贪心”的，这表示在有二义的情况下，会尽可能匹配最长的字符串；

  12. 通过在字符分类的左方括号后加上一个插入字符（^），就可以得到“非字符类”;

  13. 在正则表达式的开始处使用插入符号（^），表明匹配必须发生在被查找文本开始处，在正则表达式的末尾加上美元符号（$），表示该字符串必须以这个正则表达式的模式结束；

  14. 在正则表达式中，.（句点）字符称为“通配符”，匹配除了换行之外的所有字符；

文件

  1. 每个运行在计算机上的程序，都有一个“当前工作目录”，或 cwd，导入os，使用os.getcwd()函数可以取得当前工作路径的字符串，并可以利用 os.chdir()改变它；
  2. 用 os.makedirs()函数创建新文件夹（目录）；
  3. os.path 模块包含了许多与文件名和文件路径相关的有用函数：
    • 调用 os.path.abspath(path)将返回参数的绝对路径的字符串
    • 调用 os.path.isabs(path)，如果参数是一个绝对路径，就返回 True，否则返回False
    • 调用 os.path.relpath(path, start)将返回从 start 路径到 path 的相对路径的字符串
    • 调用 os.path.split()获得一个路径的目录名称和基本名称的元组
    • 可以调用 os.path.dirname()和 os.path.basename()获取一个路径的目录名称和基本名称
    • 调用 os.path.getsize(path)将返回 path 参数中文件的字节数
    • 调用 os.listdir(path)将返回文件名字符串的列表，包含 path 参数中的每个文件
    • 如果 path 参数所指的文件或文件夹存在，调用 os.path.exists(path)将返回 True
    • 如果 path 参数存在，并且是一个文件，调用 os.path.isfile(path)将返回 True
    • 如果 path 参数存在，并且是一个文件夹，调用 os.path.isdir(path)将返回 True
  4. shutil（或称为 shell 工具）模块中包含一些函数，让你在 Python 程序中复制、移动、改名和删除文件。要使用 shutil 的函数，首先需要 import shutil；
  5. 调用 shutil.copy(source, destination)，将路径 source 处的文件复制到路径 destination处的文件夹（source 和 destination 都是字符串）；
  6. 调用 shutil.move(source, destination)， 将路径 source 处的文件夹移动到路径 destination，并返回新位置的绝对路径的字符串；
  7. 用 os.unlink(path)将删除 path 处的文件，调用 os.rmdir(path)将删除 path 处的文件夹，要求文件夹必须为空；
  8. 调用 shutil.rmtree(path)将删除 path 处的文件夹，它包含的所有文件和文件夹都会被删除；
  9. os.walk()函数被传入一个字符串值，即一个文件夹的路径，可以在一个 for循环语句中使用 os.walk()函数，遍历目录树；
  10. zipfile 模块提供了一种方法，用 Python 压缩和解压 ZIP 归档文件

Python&Excel

  1. openpyxl 模块让 Python 程序能读取和修改 Excel 电子表格文件，要想在Python中使用openpyxl，需要安装并导入模块，使用命令import openpyxl
  2. 使用 openpyxl.load_workbook()函数接受文件名，返回一个 workbook 数据类型的值；
  3. 调用 get_sheet_names()方法可以取得工作簿中所有表名的列表 ；
  4. 调用 Workbook 对象的 get_active_sheet()方法，取得工作簿的活动表；
  5. 调用 openpyxl.cell.column_index_from_string()函数从字母转换到数字，要从数字转换到字母则调用openpyxl.cell.get_column_letter()函数；
  6. 调用 openpyxl.charts.BarChart()，创建一个条形图，调用openpyxl.charts.LineChart()、 openpyxl.charts.ScatterChart()和 openpyxl.charts.PieChart()， 创建折线图、散点图和饼图；

PDF以及Word处理

  1. 用于处理PDF的模块是PyPDF2，使用需要安装模块，然后导入import PyPDF2
    1. 导入 PyPDF2 模块 : import PyPDF2
    2. 以读二进制模式打开 name.pdf,保存在 pdfFileObj 中
    3. 调用 PyPDF2 .PdfFileReader()并向它传入 pdfFileObj ，保存返回对象在 pdfReader 中
  2. 调用 PdfFileReader 对象的 getPage()方法，传入感兴趣的页码，从而取得 Page 对象；
  3. 取得 Page 对象后，调用它的 extractText()方法，返回该页文本的字符串；
  4. 所有 PdfFileReader 对象都有一个 isEncrypted 属性，如果 PDF 是加密的，它就是 True；
  5. 要读取加密的 PDF，就调用 decrypt()函数，传入口令字符串；
  6. *PyPDF2 写入 PDF 的能力，仅限于从其他 PDF 中拷贝页面、旋转页面、重叠页面和加密文件 *
  7. 调用PdfFileWriter 对象的write()方法会实际生成文件；
  8. 利用 rotateClockwise()和 rotateCounterClockwise()方法， PDF 文档的页面也可以旋转 90 度的整数倍，向这些方法传入整数 90、 180 或 270 就可以了；
  9. 用 python-docx 模块，在 Python 中打开了一个.docx 文件，调用 docx.Document() ；

CSV和JSON数据

  1. 想要处理CSV文件，需要导入csv模块，import csv
  2. 要用 csv 模块从 CSV 文件中读取数据，需要创建一个 Reader 对象， Reader 对象可以遍历CSV中的每一行；
    1. 用 open()函数打开它
    2. 将它传递给 csv.reader()函数，返回一个 Reader 对象
    3. 要访问 Reader 对象中的值，最直接的方法，就是将它转换成一个普通 Python 列表， 即将它传递给 list()
  3. Writer 对象可以将数据写入 CSV 文件，使用csv.writer()函数创建改对象；
    1. 调用 open()并传入’w’，以写模式打开一个文件
    2. 将它传递给 csv.writer() ，创建一个 Writer 对象
  4. Writer 对象的 writerow()方法接受一个列表参数 ，列表中的每个词，放在输出的CSV 文件中的一个单元格中；
  5. JSON处理需要导入模块，import json，要将包含 JSON 数据的字符串转换为 Python 的值，就将它传递给 json.loads()函数，json.dumps()函数将一个 Python 值转换成 JSON 格式的数据字符串；

图形处理

  1. Pillow 是一个第三方 Python 模块， 用于处理图像文件，该模块包含一些函数，可以很容易地裁剪图像、调整图像大小，以及编辑图像的内容；

  2. 计算机程序通常将图像中的颜色表示为 RGBA 值，指定顔色中的红、绿、蓝和 alpha（透明度）的值；

  3. 像素是计算机屏幕上能显示一种顔色的最小点，像素的 RGB 设置准确地告诉它应该显示哪种颜色的色彩；

  4. Pillow 提供 ImageColor.getcolor()函数：

  5. 许多 Pillow 函数和方法需要一个矩形元组参数；

  6. 要加载图像，就从 Pillow 导入 Image 模块，并调用 Image.open()，传入图像的文件名，返回值是 Image 对象数据类型，通过 save()方法，对 Image 对象的所有更改都可以保存到图像文件中（也是任何格式）；

  7. 所有的旋转、调整大小、裁剪、绘画和其他图像操作，都通过这个 Image 对象上的方法调用来完成；

  8. Image.new()返回的对象表示空白的图像，Image.new()的参数如下：

    • 字符串’RGBA’，将颜色模式设置为 RGBA
    • 图像开始采用的背景颜色，是一个表示 RGBA 值的四整数元组

  9. Image 对象的 crop()方法接受一个矩形元组，返回一个 Image 对象，表示裁剪后的图像，裁剪不是在原图上发生的；

  10. copy()方法返回一个新的 Image 对象，它和原来的 Image 对象具有一样的图像；

  11. paste()方法在 Image 对象调用，将另一个图像粘贴在它上面；

  12. resize()方法在 Image 对象上调用，返回指定宽度和高度的一个新 Image 对象，resize()方法不会在原图上修改 Image 对象，而是返回一个新的 Image 对象；

  13. rotate()方法旋转，该方法返回旋转后的新 Image 对象，并保持原始Image 对象不变。rotate()的参数是一个整数或浮点数，表示图像逆时针旋转的度数；

  14. transpose()方法，还可以得到图像的“镜像翻转”，必须向 transpose()方法传入 Image.FLIP_LEFT_RIGHT 或 Image.FLIP_TOP_BOTTOM；

  15. 单个像素的颜色可以通过 getpixel()和 putpixel()方法取得和设置；

  16. ImageDraw 对象：

    1. text()方法，用于在图像上绘制文本
    2. point(xy, fill)方法绘制单个像素。xy 参数表示要画的点的列表；
    3. line(xy, fill, width)方法绘制一条线或一系列的线；
    4. rectangle(xy, fill, outline)方法绘制一个矩形；
    5. ellipse(xy, fill, outline)方法绘制一个椭圆；
    6. polygon(xy, fill, outline)方法绘制任意的多边形；

时间、计划任务

  1. 内置的 time 模块让 Python程序能读取系统时钟的当前时间，在 time 模块中， time.time()和 time.sleep()函数是最有用的模块，使用模块时导入import time；
  2. 需要让程序暂停一下，就调用 time.sleep()函数，并传入希望程序暂停的秒数；
  3. datetime 模块有自己的 datetime 数据类型；
  4. datetime 模块还提供了 timedelta 数据类型，它表示一段时间，而不是一个时刻；
  5. 利用内建的 subprocess 模块中的 Popen()函数， Python 程序可以启动计算机中的其他程序；
  6. 如果想在 Python 脚本中启动一个外部程序，就将该程序的文件名传递给subprocess.Popen() ；

GUI模拟鼠标键盘

  1. pyautogui 模块包含了一些函数，可以模拟鼠标移动、按键和滚动鼠标滚轮；
  2. pyautogui 的鼠标函数使用 x、 y 坐标，pyautogui.size() 函数返回两个整数的元组，包含屏幕的宽和高的像素数 ；
  3. pyautogui.moveTo() 函数将鼠标立即移动到屏幕的指定位置；
  4. pyautogui.moveRel() 函数相对于当前的位置移动鼠标接受 3 个参数：向右水平移动多少个像素，向下垂直移动多少个像素，以及（可选的）花多少时间完成移动；
  5. 调用 pyautogui.position() 函数，可以确定鼠标当前的位置，返回函数调用时，鼠标 x、 y 坐标的元组；
  6. 要向计算机发送虚拟的鼠标点击，就调用 pyautogui.click() 方法，pyautogui.click（100， 150， button=’left’）；
  7. pyautogui.doubleClick() 函数只执行双击鼠标左键，pyautogui.rightClick() 和 pyautogui.middleClick() 函数将分别执行双击右键和双击中键；
  8. PyAutoGUI 提供了 pyautogui.dragTo() 和 pyautogui.dragRel() 函数，将鼠标拖动到一个新的位置，或相对当前位置的位置。 dragTo() 和 dragRel() 的参数与moveTo() 和 moveRel() 相同；
  9. scroll()是滚动鼠标函数，单位的意义在每个操作系统和应用上不一样；
  10. pyautogui 拥有屏幕快照的功能，可以根据当前屏幕的内容创建图形文件，调用 pyautogui.screenshot() 函数 获取屏幕快照；
  11. pyautogui.typewrite() 函数向计算机发送虚拟按键，按键产生什么效果，取决于当前获得焦点的窗口和文本输入框；
  12. pyautogui.keyDown() 和 pyautogui.keyUp() 将向计算发送虚拟的按键和释放，pyautogui 提供了pyautogui.press() 函数，它调用这两个函数，模拟完整的击键；
  13. pyautogui.hotkey() 函数，它接受多个键字符串参数，按顺序按下，再按相反的顺序释放；

  3. Beautiful Soup：解析 HTML，即网页编写的格式
