社会工程—基础入门

发表于 2020-06-21 | 分类于 社会工程

简介

只有了解各种钓鱼欺诈，才能更好地防范这些诈骗。了解钓鱼欺诈中所使用的心理学原则和技术工具， 可以辨识和防范各种类型和难度级别的钓鱼欺诈！几乎每一起成功的网络攻击的核心都是人的因素！推荐书籍：《社会工程·防范钓鱼欺诈（卷3）》· [美] Christopher·Hadnagy Michele·Fincher 著 肖诗尧 译

开始

基础知识

  1. 几乎所有的大型攻击事件的分析结果表明，绝大多数攻击都与社会工程有关，涉及网络钓鱼邮件（ phishing e-mail）、鱼叉式网络钓鱼（ spear phish）或恶意来电（ malicious phone call， vishing）；
  2. 各种形式的网络钓鱼已经成为一种备受关注的攻击手段，因为这是让人们不假思索行事的简单方法；
  3. 鱼叉式网络钓鱼是一种非常有针对性的攻击方式。攻击者花时间对目标进行研究，然后创建与目标个人信息相关的或者私人化的电子邮件；
  4. 骗子会给刚收到钓鱼邮件的人打电话。这也被称为语音钓鱼（ vishing）或电话钓鱼；
  5. 钓鱼成功的原因：
    • 贪婪
    • 缺乏教育
    • 过于轻信
  6. 网络钓鱼并不只是关于你点击了什么，而是关于你为什么会点击它！

心理学原则

  1. 决策的好坏并不总是与我们对这个决策本身是否感到满意有关；

  2. 决策包含了一系列因素，如我们的看法和情绪；

  3. 我们每天都在做出或大或小的决策，即使在并不具备所需的全部相关信息的情况下；

  4. 我们不假思索地、频繁地做出这些或大或小的决策；

  5. 认知偏差是一种思维定势，通常源于过去的经验！

  6. 框架效应（ framing effect）就是一种认知偏差，它指的是你的反应取决于当时的情形，或者说你的反应依赖于语境；

  7. 确认偏误（ confirmation bias），你倾向于寻找支持你的观点的信息，或者按照支持你的观点的方式来解读信息；

  8. 研究人员发现，一夜未眠会导致做出更冒险的决策；

  9. 人类有一种服从别人的倾向，尤其是在特定的情况下！

  10. 钓鱼攻击者了解我们是如何决策的，并且试图操纵我们所面临的情况以引导我们做出错误决定；

  11. 情绪反应是正常的，你要做的只是调整一下，这样你就可以避免一些日后会让你后悔的行为；

  12. 反思自己的决策：

    • 确保正确地理解问题
    • 尽可能完整地收集信
    • 考虑切实可行的选项
    • 做出决策
    • 评估效果

影响与操控

  1. 影响（ influence）和操控（ manipulation） 都是指一个人采取了对另一个人生效的行为；

  2. 影响显然是积极的，操控则是消极的；

  3. 建立融洽的关系是成功的社会工程人员为了提高效率而必须快速培养的技能；

  4. 人类和其他物种似乎有着天生的欺骗的倾向！

  5. 欺骗是一种社交行为的适应性形式，没有必要总是把它想得那么负面；

  6. 通过惩罚来改变某个成年人的行为是完全可能的，但这并不是高明的暗中操纵，它只是简单的能产生某个后果的行为；

自我保护

  1. 黑客会攻击全球范围内的真实的服务器，并用合法服务器来发送钓鱼邮件；
  2. 恶意钓鱼攻击者还有很多引诱受害者点击链接的手段；
  3. 病毒的指纹可以用来识别病毒，病毒的指纹被放进了反病毒软件的数据库中，然后反病毒软件对你的电脑进行扫描；
  4. 坏人更高明、坚定和努力。他们似乎领先于你，并依靠你的弱点取胜；
  5. 仅仅依靠技术是救不了你的，技术也不应该被视作社会工程学的解决方案；

钓鱼攻击工具包

  1. Rapid7 Metasploit Pro可以通过闭环漏洞验证来显示风险程度并进行风险优先级划分，还能通过模拟钓鱼邮件攻击来对安全意识进行衡量；
  2. ThreatSim允许组织机构根据终端用户的行为评估和降低风险；
  3. PhishMe是一种 SaaS 解决方案，利用浸入式教学法来对员工进行识别和规避钓鱼攻击的培训；
  4. Wombat PhishGuru模拟钓鱼攻击服务，你可以通过发送模拟钓鱼邮件来评估你的员工易受攻击的程度并鼓励他们加强训练；
  5. PhishLine是一种企业级的 SaaS 解决方案，可模拟真实世界的社会工程学和钓鱼攻击，提供在线安全意识训练、基于风险的调查以及详细的报告和指标；
  6. SET为评估者提供了一种测试其培训和安全意识项目有效性的机制；
  7. Phishing Frenzy是一款由 Ruby on Rails 构建的基于 Linux的开源应用，也是渗透测试者用于管理钓鱼邮件攻击的工具；

总结

这世上没有公平竞争这一回事，要利用一切弱点！学校先教你知识然后考试，生活则通过考试来教你知识。安全永远最薄弱的环境当属人参与的环节！社会工程就是利用人的弱点来进行攻击！
